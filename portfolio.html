Modulated light communication via LED signages and its detection on cellphones can be very versatile in many applications, but sometimes they suffer from luminance defects and bringing in some AI may help. To this end, I developed pixel-wise LED signage brand object detection using an open-source SSD + MobileNetV2 model on the Tensorflow-Lite platform to alleviate this issue. This AI object detection demo was implemented on a low-end Android phone, Samsung Galaxy S4 in real time.
